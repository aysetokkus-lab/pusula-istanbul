PUSULA REHBER — DEVAM NOTU (23 Mart 2026)
==========================================

Asagidaki metni yeni oturumda yapistur:

---

Bu proje "Pusula Rehber" — profesyonel Istanbul turist rehberleri icin React Native Expo uygulamasi. Kaldigimiz yerden devam ediyoruz.

TEKNOLOJI:
- React Native Expo (SDK 54, React 19.1, Expo Router 6.0.23)
- expo-linear-gradient, @expo-google-fonts/poppins, expo-image, expo-font, expo-splash-screen
- Supabase (auth + real-time data)
- Light mode aktif (hooks/use-tema.ts → isDark = false)

PROJE YAPISI:
app/(tabs)/
  _layout.tsx    — 4 tab: Ana Sayfa, Acil, Ara, Profil (+ muzeler, bogaz, ulasim, muzeKart gizli)
  index.tsx      — Ana sayfa (~580 satir) — tum gradient bantlar, 8li ikon grid, modaller
  acil.tsx       — Acil numaralar, hesap
  ara.tsx        — Arama ekrani (placeholder)
  profil.tsx     — Profil ekrani (placeholder)
  muzeler.tsx    — 32 mekan, kategori, detay modal
  bogaz.tsx      — Vapur sefer saatleri
  ulasim.tsx     — Havalimani ulasim
  muzeKart.tsx   — Muze Kart bilgileri

components/
  canli-durum-panel.tsx — useSahaBildir hook + CanliDurumOzet + DurumBildirModal

constants/theme.ts — Palette, Space, Radius, TemaRenkleri, light/dark

hooks/
  use-tema.ts        — { t, isDark }
  use-canli-durum.ts — Supabase real-time saha bildirimleri

assets/icons/ — 8 SVG (beyaz line-art): namaz-vakitleri, muzeler, saraylar, bogaz-turlari, havalimani-ulasim, gemi-tarihleri, doviz-kuru, logo

ANA SAYFA (index.tsx) YAPISI (yukaridan asagiya):
1. Header — pusula [logo] istanbul, mavi diagonal gradient ['#00A8E8', '#0077B6', '#0096C7', '#48CAE4'], Poppins font, logo ortada, yazilar flex:1 ile simetrik
2. Greeting Strip — Ayni mavi gradient, saat (sol), "Hos geldin" + canli hava durumu (wttr.in API) + "Iyi turlar" (sag), tum yazilar beyaz
3. Saha Durumu bandi — Ayni mavi gradient, "+ Bildir" butonu → useSahaBildir hook ile DurumBildirModal aciliyor
4. Sultan Ahmet Camii bandi — Ayni mavi gradient, tiklaninca Sultanahmet modal aciliyor, sagda ACIK/KAPALI durumu
5. Yaklasan Etkinlikler bandi — Sari-turuncu-kirmizi diagonal gradient ['#FFD60A', '#FF9F1C', '#E63946', '#C1121F']
6. Ulasim Uyarilari bandi — Ayni sari-kirmizi gradient
7. 8li Ikon Grid (4x2) — Gradient mavi daireler, SVG ikonlar (44x44), sira: Namaz, Muzeler, Saraylar, Bogaz, Havalimani, Muze Kart, Gemi, Doviz
8. Bugunku Gemi Listesi bandi — Ayni sari-kirmizi gradient + gemi karti/modal
- Tum bantlar arasi 3px beyaz whiteSeparator
- 3 Modal: Gemi Takvimi, Sultanahmet, Ayasofya
- Canli hava durumu: wttr.in API, emoji ikon esleme
- Sultanahmet ziyaret pencereleri: namaz vakitlerine gore dinamik hesaplama
- Galataport gemi takvimi: Mart-Mayis 2026 verileri
